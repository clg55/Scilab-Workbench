Scilab Function

NOM

    plot2d3 - représentation d'une courbe sous forme de barres verticales.

SEQUENCE D'APPEL

plot2d3([logflags,] x,y,[style,strf,leg,rect,nax])
plot232(y)
plot2d3(x,y <,opt_args>)

PARAMETRES

    [ ] : voir plot2d pour une description des paramètres.

DESCRIPTION

    plot2d3 est identique à plot2d mais les courbes sont dessinées sous forme de barres verticales.

    Par défaut, les dessins successifs sont superposés. Pour effacer le dessin précédent, utiliser xbasc() .

    Taper plot2d3() pour voir une démonstration de la fonction.

EXEMPLES

    // à comparer avec plot2d
x=[0:0.1:2*%pi]';
plot2d3(x,[sin(x) sin(2*x) sin(3*x)])

VOIR AUSSI
