Scilab Function

NOM

    xend - termine la session graphique courante

SEQUENCE D'APPEL

xend()

DESCRIPTION

    xend est utilisée pour terminer une session graphique. Avec les pilotes PostScript, XFig ou Gif, xend ferme le fichier qui a été ouvert par xinit .

EXEMPLES

    driver("Pos")
xinit("foo.ps")
plot2d()
xend()
driver("X11")

VOIR AUSSI

    xbasimp xend
