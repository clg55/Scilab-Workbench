Scilab Function

NOM

    xlfont - charge une police ou donne la liste des polices chargées dans le contexte graphique

SEQUENCE D'APPEL

xlfont(font-name,font-id)
fonts=xlfont()

PARAMETRES

    • font-name : chaîne de caractères, nom de famille de la police
    • font-id : entier entre 0 et 6.
    • fonts : vecteur colonne des noms des polices déjà présentes

DESCRIPTION

    Sans argument, xlfont() renvoie la liste des polices déjà présentes.

    Avec ses arguments d'entrée, xlfont est utilisée pour charger une police à différentes tailles dans le contexte graphique. La police doit être disponible en tailles 8, 10, 12, 14, 18 et 24.

    Les polices par défaut sont "Courier Roman" (0), "Symbol" (1), "Times Roman" (2), "Times Italic" (3), "Times Bold" (4) et "Times Bold Italic" (5).

    font-name peut être de 2 types :

  • - si il contient le caractère "%", la police est supposée être un nom de police X11 avec %s dans le champ de taille du nom de la police, par exemple
  • - s'il ne contient pas le caractère "%", il est supposé être un alias et les polices d'alias sont chargées.
  • font-id : donne le code à la police chargée .
