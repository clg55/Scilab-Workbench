Scilab Function

NOM

    driver - sélectionne un pilote graphique

SEQUENCE D'APPEL

driver(driver_name)
current_driver=driver()

PARAMETRES

    • driver_name : chaîne de caractères, le nom du pilote à sélectionner

DESCRIPTION

    Cette fonction est utilisée pour sélectionner un pilote graphique, ou sans argument pour récupérer le nom du pilote courant. La plupart du temps, l'utilisateur peut ignorer cette fonction et changer le pilote par appel à des fonctions de haut niveau telles que xbasc ou xbasimp . Le nom du pilote peut prendre les valeurs :

  • "X11" : sortie sur l'écran
  • "Pos" : sortie dans un fichier PostScript
  • "Rec" : sortie à l'écran avec enregistrement des commandes graphiques. C'est le pilote par défaut.
  • "Fig" : sortie dans un fichier au format XFig. Avec ce format le clipping n'est pas supporté.
  • "GIF" : sortie dans un fichier au format GIF (pilote en version beta fourni par Tom Leitner, http://wiis.tu-graz.ac.at/people/tom.html). L'épaisseur des lignes n'est pas prise en compte.

VOIR AUSSI

    xtape xbasc xbasimp
