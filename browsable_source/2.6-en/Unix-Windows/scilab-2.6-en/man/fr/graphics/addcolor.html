Scilab Function

NOM

    addcolor - ajoute de nouvelles couleurs à la table des couleurs courante

SEQUENCE D'APPEL

new=addcolor(c)

PARAMETRES

    • new : indices des nouvelles couleurs dans la nouvelle table des couleurs
    • c : matrice à trois colonnes. Définition des composantes RVB des nouvelles couleurs

DESCRIPTION

    addcolor ajoute de nouvelles couleurs données dans l'argument c . c est une matrice à trois colonnes. [R V B] R est la composante rouge, V est la composante verte, B est la composante bleue. Chaque terme de c est un nombre positif inférieur ou égal à un.

    Les indices des nouvelles couleurs sont renvoyés dans new .

    Si une couleur dans c est déjà présente dans la table des couleurs courante, elle n'est pas ajoutée.
