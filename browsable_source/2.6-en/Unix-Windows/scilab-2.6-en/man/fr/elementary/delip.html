Scilab Function

NOM

    delip - intégrale elliptique

SEQUENCE D'APPEL

[r]=delip(x,ck)

PARAMETRES

    • x : réel (ou vecteur réel)
    • ck : scalaire entre -1 et 1
    • r : réel ou complexe (ou vecteur)

DESCRIPTION

    Renvoie la valeur de l'intégrale elliptique avec le paramètre ck :

    x réel et positif. Quand x est un vecteur r est évalué pour chaque composante de x .

EXEMPLES

    ck=0.5;
delip([1,2],ck)
deff('y=f(t)','y=1/sqrt((1-t^2)*(1-ck^2*t^2))')
intg(0,1,f)    // OK puisque la solution est réelle !
