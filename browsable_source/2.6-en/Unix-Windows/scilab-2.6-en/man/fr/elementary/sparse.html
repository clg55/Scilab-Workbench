Scilab Function

NOM

    sparse - initialisation d'une matrice creuse

SEQUENCE D'APPEL

sp=sparse(X)
sp=sparse(ij,v [,mn])

PARAMETRES

    • X : matrice réelle ou complexe (pleine ou creuse)
    • ij : matrice d'entiers à deux colonnes (indices des termes non-nuls)
    • mn : vecteur d'entiers à deux composantes (nombre de lignes, nombre de colonnes)
    • sp : matrice creuse

DESCRIPTION

    sparse est utilisée pour initialiser une matrice creuse. Seuls les termes non-nuls sont stockés.

    sp = sparse(X) convertit une matrice pleine en une matrice creuse en supprimant les éléments nuls (si X est déjà stockée comme une matrice creuse, sp est égale à X ).

    sp=sparse(ij,v [,mn]) construit une matrice creuse mn(1) par mn(2) avec sp(ij(k,1),ij(k,2))=v(k) . ij et v doivent avoir le même nombre de colonnes. Si le paramètre optionnel mn n'est pas donné les dimensions de la matrice sp sont égales aux valeurs maximales de ij(:,1) et de ij(:,2) , respectivement.

    Les opérations (concaténation, addition, etc.) avec les matrices creuses se font avec la même syntaxe que pour les matrices pleines.

    Les fonctions élémentaires ( abs,maxi,sum,diag,... ) sont aussi disponibles pour les matrices creuses.

    Les opérations mixtes (matrice pleine - matrice creuse) sont autorisées. Le résultat est une matrice pleine ou creuse suivant l'opération.

EXEMPLES

    sp=sparse([1,2;4,5;3,10],[1,2,3])
size(sp)
x=rand(2,2);abs(x)-full(abs(sparse(x)))
