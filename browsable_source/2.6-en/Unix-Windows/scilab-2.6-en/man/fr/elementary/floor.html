Scilab Function

NOM

    floor - partie entière

SEQUENCE D'APPEL

[y]=floor(x)

PARAMETRES

    • x : matrice réelle
    • y : matrice entière

DESCRIPTION

    Renvoie dans y pour chaque élément de x sa partie entière (l'entier directement inférieur).

EXEMPLES

    floor([1.9 -2.5])-[2,-2]
floor(-%inf)
x=rand()*10^20;floor(x)-x

VOIR AUSSI

    round floor int
