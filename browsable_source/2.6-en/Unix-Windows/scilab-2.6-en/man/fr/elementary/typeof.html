Scilab Function

NOM

    typeof - type d'un objet

SEQUENCE D'APPEL

[t]=typeof(obj)

PARAMETRES

    • obj : objet Scilab
    • t : string

DESCRIPTION

    t=typeof(obj) renvoie une des chaînes de caractères :

  • "constant" si l'objet est une matrice réelle ou complexe
  • "polynomial" si l'objet est une matrice de polynômes
  • "function" si l'objet est une fonction
  • "string" si l'objet est une matrice de chaînes de caractères
  • "boolean" si l'objet est une matrice booléenne
  • "list" si l'objet est une liste
  • "rational" si l'objet est une matrice de fractions rationnelles
  • "state-space" si l'objet est un système dynamique linéaire (voir )
  • "sparse" si l'objet est une matrice creuse
  • "boolean sparse" si l'objet est une matrice booléenne creuse

EXEMPLES

    typeof(1)
typeof(poly(0,'x'))
typeof(1/poly(0,'x'))
typeof(%t)
w=sprand(100,100,0.001);
typeof(w)
typeof(w==w)
deff('y=f(x)','y=2*x');
typeof(f)
