Scilab Function

NOM

    binomial - distribution binomiale

SEQUENCE D'APPEL

pr=binomial(p,n)

PARAMETRES

    • pr : vecteur, pr(k) = probabilité (X=k) , avec X=B(n,p).
    • p : scalaire
    • n : entier

DESCRIPTION

    pr=binomial(p,n) renvoie un vecteur de probabilités binomiales. pr(k) = probabilité (X=k), avec X=B(n,p) c'est à dire la probabilité de k succès lors de n essais indépendants, (p = probabilité de succès). pr(k) = C(n,k) p^k (1-p)^(n-k) avec C(n,k)= n!/(k!(n-k)!) = prod(1:n)/(prod(1:k)*prod(1:n-k)).

EXEMPLES

    n=10;p=0.3;plot2d3(0:n,binomial(p,n));
n=100;p=0.3;
mea=n*p;sigma=sqrt(n*p*(1-p));
x=( (0:n)-mea )/sigma;
plot2d(x,sigma*binomial(p,n));
deff('y=Gauss(x)','y=1/sqrt(2*%pi)*exp(-(x.^2)/2)')
plot2d(x,Gauss(x));
