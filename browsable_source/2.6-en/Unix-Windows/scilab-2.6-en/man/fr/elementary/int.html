Scilab Function

NOM

    int - arrondi vers zéro ou troncature

SEQUENCE D'APPEL

[y]=int(X)

PARAMETRES

    • X : matrice réelle
    • y : matrice d'entiers

DESCRIPTION

    int(X) renvoie une matrice d'entiers dont les éléments sont les arrondis vers zéro des éléments de x. Cette fonction est la même que la fonction fix .

VOIR AUSSI

    round floor ceil fix
