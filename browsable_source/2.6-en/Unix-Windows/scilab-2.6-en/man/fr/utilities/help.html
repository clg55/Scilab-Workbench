Scilab Function

NOM

    help - commande d'aide en ligne

SEQUENCE D'APPEL

help <mot-clé>

DESCRIPTION

    A chaque mot-clé documenté correspond un fichier HTML généralement appelé mot-clé.html . Ce fichier HTML vient de la traduction d'un fichier XML correspondant. Ces fichiers sont organisés dans des répertoires (ou chapitres). Chaque répertoire contient des fichiers *.html et un fichier whatis.html contenant une ligne pour chaque mot-clé documenté dans le chapitre correspondant. Chaque ligne doit avoir le format : 

    <BR><A HREF="mot-clé.html">mot-clé</A> - description rapide

    La liste des chapitres de l'aide est contenue dans la variable Scilab %helps . %helps est une matrice de chaînes de caractères à deux colonnes. Chaque ligne est formée comme ceci 

    répertoire_du_chapitre    titre_du_chapitre

    La valeur par défaut de %helps est initialisée dans le fichier SCI+"scilab.star" .
