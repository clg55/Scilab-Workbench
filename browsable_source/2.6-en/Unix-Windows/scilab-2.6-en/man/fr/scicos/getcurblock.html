Scicos function

NAME

    getcurblock - get Scicos current block number while running

PARAMETERS

    • blk : the index of the Scicos block which has called getcurblock

DESCRIPTION

    This function may be used in a Scilab block to get index of this block while running. It may be used with getscicosvars or setscicosvars .

SEE ALSO

    getscicosvars setscicosvars scicosim
