Scilab Toolbox

NAME

    stixbox - Stixbox Toolbox

DESCRIPTION

    Stixbox is a statistics toolbox based on the toolbox by Anders Holtsberg with extensions from the Paris Sud University.

    The Scilab toolbox stixbox-1.2 is derived from the matlab stixbox 1.1 version.

    IMPORTANT NOTE: you need to load the stixbox before using it by executing in Scilab the command load_stixbox() .
