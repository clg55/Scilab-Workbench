Scilab Function

NAME

    playsnd - sound player facility

CALLING SEQUENCE

[]=playsnd(y,fs,bits)

PARAMETERS

    • y : real vector
    • fs : real number, sampling frequency
    • bits : real number, number of bits (usually 8 or 16)

DESCRIPTION

    Redirects a linear signal to /dev/audio/ .
