Scilab Function

NOM

    sqroot - factorisation hermitienne W*W'

SEQUENCE D'APPEL

sqroot(X)

PARAMETRES

    • X : matrice complexe ou réelle, symétrique définie non-négative

DESCRIPTION

    renvoie W telle que X=W*W' (en utilisant svd).

EXEMPLES

    X=rand(5,2)*rand(2,5);X=X*X';
W=sqroot(X)
norm(W*W'-X,1)
//
X=rand(5,2)+%i*rand(5,2);X=X*X';
W=sqroot(X)
norm(W*W'-X,1)

VOIR AUSSI

    chol svd
