Scilab Function

NOM

    spec - valeurs propres d'une matrice

SEQUENCE D'APPEL

evals=spec(A)

PARAMETRES

    • A : matrice carrée réelle ou complexe
    • evals : vecteur réel ou complexe

DESCRIPTION

    renvoie dans le vecteur evals les valeurs propres de A .

    Les vecteur propres peuvent être obtenus avec bdiag .

EXEMPLES

    A=diag([1,2,3]);X=rand(3,3);A=inv(X)*A*X;
spec(A)
//
x=poly(0,'x');
pol=det(x*eye()-A)
roots(pol)
//
[Ab,X,bs]=bdiag(A);
Ab
clean(inv(X)*A*X)
