Scilab Function

NOM

    rcond - estimation de l'inverse du conditionnement

SEQUENCE D'APPEL

rcond(X)

PARAMETRES

    • X : matrice carrée réelle ou complexe

DESCRIPTION

    est une estimation de l'inverse du conditionnement de X pour la norme l_1.

    Si X est bien conditionnée, rcond(X) est proche 1. Sinon, rcond(X) est proche de 0.

    renvoie rcond(X) dans r et renvoie aussi z tel que

    Ainsi, si rcond est très petit z est un vecteur se trouvant dans le noyau de X.

EXEMPLES

    A=diag([1:10]);
rcond(A)
A(1,1)=0.000001;
rcond(A)
