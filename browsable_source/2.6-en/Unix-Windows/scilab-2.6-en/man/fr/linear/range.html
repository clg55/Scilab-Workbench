Scilab Function

NOM

    range - Image de A^k

SEQUENCE D'APPEL

[X,dim]=range(A,k)

PARAMETRES

    • A : matrice réelle carrée
    • k : entier
    • X : matrice réelle régulière
    • dim : entier (dimension du sous-espace)

DESCRIPTION

    Calcul de l'image de A^k ; les dim premières colonnes de X forment une base de A^k .

VOIR AUSSI

    fullrfk rowcomp
