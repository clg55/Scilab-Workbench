Scilab Function

NOM

    pause - mode de pause, attente clavier

DESCRIPTION

    Passage en mode de pause ; inséré dans le code d'une fonction, pause fait une pause dans l'exécution de la fonction : un symbole d'invite de commande est affiché à l'utilisateur indiquant le "niveau" de la pause (e.g. -1-> ). L'utilisateur se trouve dans un environnement dans lequel toutes les variables du niveau correspondant sont accessibles (en particulier les variables locales de la fonction). Pour reprendre l'exécution taper "return" .

    Dans ce mode, [...]=return(...) renvoie les variables de l'argument (...) dans l'environnement appelant sous les noms précisés dans [...] . Sinon, les variables de l'environnement interrompu sont protégées et non modifiables.

    L'instruction pause est très utile en phase de mise au point ou de déboggage.

    Il est possible de sortir du mode de pause en mettant fin à l'exécution en tapant "abort" .
