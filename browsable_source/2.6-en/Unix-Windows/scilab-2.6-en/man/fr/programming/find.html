Scilab Function

NOM

    find - recherche de termes %T (vrai) dans une matrice de booléens

SEQUENCE D'APPEL

[ii]=find(x)
[ir,ic]=find(x)

PARAMETRES

    • x : matrice de booléens, ou matrice d'un autre type standard
    • ii, ir, ic : vecteurs d'entiers ou vecteurs vides

DESCRIPTION

    Si x est une matrice de booléens,

    ii=find(x) renvoie le vecteur des indices i pour lesquels x(i) est %T="vrai". Si aucun élément "vrai" n'est trouvé la fonction find renvoie une matrice vide.

    [ir,ic]=find(x) renvoie deux vecteurs d'indices ir (pour les lignes) et ic (pour les colonnes) tels que x(ir(n),ic(n)) est "vrai". Si aucun élément "vrai" n'est trouvé la fonction find renvoie ir et ic vides.

    Si x est une matrice d'un autre type standard find(x) est interprété comme find(x<>0)

    find([]) renvoie []

EXEMPLES

    A=rand(1,20);
w=find(A<0.5);
A(w)
w=find(A>100);

VOIR AUSSI

    boolean extraction insertion
