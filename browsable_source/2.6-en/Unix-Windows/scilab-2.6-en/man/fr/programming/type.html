Scilab Function

NOM

    type - type d'une variable

SEQUENCE D'APPEL

[i]=type(x)

PARAMETRES

    • x : objet Scilab
    • i : entier

DESCRIPTION

    type(x) renvoie un entier donnant le code numérique du type de x définit comme suit :

    1 : matrice réelle ou complexe

    2 : matrice polynomiale

    4 : matrice booléenne

    5 : matrice creuse

    8 : matrice d'entiers codés sur 1, 2 ou 4 octets

    10 : matrice de chaînes de caractères

    11 : fonction non-compilée

    13 : fonction compilée

    14 : fonction d'une librairie

    15 : liste

    16 : liste typée (tlist)

    128 : pointeur

VOIR AUSSI

    typeof
