Scilab Function

NOM

    while - mot-clé utilisé dans une structure while ... end

DESCRIPTION

    Mot-clé while . Doit être terminé par "end"

    Notes:

  • - La seule contrainte est que chaque mot-clé ou soit sur la même ligne que le correspondant.
  • - Les mot-clés et peuvent être remplacés par un passage à la ligne ou une virgule.
  • - La construction permet de spécifier des instructions à exécuter si l'expression est fausse.

    • Attention : le nombre de caractères constituant le corps d'une instruction conditionnelle (if while for ou select/case) est limité à 16k..

EXEMPLES

    e=1; a=1; k=1;
while norm(a-(a+e),1) > %eps, e=e/2; k=k+1; end
e,k

VOIR AUSSI

    for select break return pause
