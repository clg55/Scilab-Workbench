Scilab Function

NOM

    getenv - renvoie la valeur d'une variable d'environnement

SEQUENCE D'APPEL

env=getenv(str [, rep] )

PARAMETRES

    • str : chaîne de caractères contenant le nom de la variable d'environnement rep : chaîne de caractères optionnelle. Quand elle est utilisée, getenv renvoie la valeur rep quand la variable d'environnement str n'existe pas.
    • env : chaîne de caractères contenant la valeur de la variable d'environnement

DESCRIPTION

    Renvoie la la valeur d'une variable d'environnement, quand celle-ci existe.

EXEMPLES

     getenv('SCI')
 getenv('FOO','foo')
