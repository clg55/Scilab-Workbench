Scilab Function

NOM

    equal (=) - affectation, comparaison

DESCRIPTION

    Le signe d'égalité est utilisé pour affecter une expression à une variable

    == est l'opérateur de comparaison entre deux expressions et renvoie une matrice booléenne.

EXEMPLES

    a=sin(3.2)
[u,s]=schur(rand(3,3))
[1:10]==4
1~=2

VOIR AUSSI

    less boolean
