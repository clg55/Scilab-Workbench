Scilab Function

NOM

    iserror - test d'occurrence d'une erreur

SEQUENCE D'APPEL

iserror([n])

DESCRIPTION

    Vérifie si l'erreur n a eu lieu (après un appel à errcatch ). iserror renvoie 1 si l'erreur a eu lieu et 0 sinon.

    n>0 est le numéro de l'erreur; toutes les erreurs sont testées si n<0 .

VOIR AUSSI

    error errcatch
