Scilab Function

NOM

    format - format d'affichage des nombres

SEQUENCE D'APPEL

format([type],[long])
format()

PARAMETRES

    • type : chaîne de caractères
    • long : entier (nombre maximum de chiffres, 10 par défaut)

DESCRIPTION

    Change le format d'affichage courant selon la valeur du paramètre type , qui peut prendre une des valeurs suivantes :

  • "v" : format variable (par défaut)
  • "e" : format avec exposant

    • long définit le nombre maximum de chiffres (10 par défaut). format() renvoie un vecteur précisant le format courant : la première composante est le type du format (0 pour 'v' ; 1 pour 'e' ); la deuxième composante est le nombre de chiffres.

    En mode format variable, les composantes plus petites que %eps fois la composante maximum en valeur absolue étaient affichées "0" dans les versions précédentes de Scilab. Ce n'est plus le cas, la fonction clean peut être utilisée pour forcer à zéro les composantes négligeables.

EXEMPLES

    x=rand(1,5);
format('v',10);x
format(20);x
format('e',10);x
format(20);x

x=[100 %eps];
format('e',10);x
format('v',10);x

format()
