Scilab Function

NOM

    lasterror - dernier numéro d'erreur enregistré

SEQUENCE D'APPEL

str=lasterror( [opt] )
[str,n]=lasterror([opt])

PARAMETRES

    • str : vecteur de chaînes de caractères ou chaîne vide. Le dernier message d'erreur enregistré.
    • n : un entier, 0 ou le numéro de la dernière erreur enregistrée.
    • opt : un booléen, %T si le message d'erreur doit être effacé (%T par défaut)

DESCRIPTION

    À chaque fois qu'une erreur se produit, le système de gestion d'erreur de Scilab l'enregistre dans une table interne (uniquement la dernière erreur est enregistrée). La fonction lasterror permet d'obtenir le numéro ainsi que la chaîne de caractères du message associé à la dernière erreur.

    Cette fonction est utilisée conjointement avec les fonctions errcatch ou execstr .

    Le message et le numéro d'erreur peuvent être conservés pour un appel ultérieur à lasterror en utilisant la syntaxe lasterror(%f) .

EXEMPLES

    ierr=execstr('a=zzzzzzz','errcatch')
if ierr>0 then disp(lasterror()),end

VOIR AUSSI

    errcatch execstr error errclear
