Scilab command

NOM

    pwd - affiche le répertoire courant de Scilab

    getcwd - renvoie le répertoire courant de Scilab

SEQUENCE D'APPEL

pwd
x=pwd()
x=getwcd()

DESCRIPTION

    pwd renvoie dans ans le répertoire courant de Scilab. x=pwd() ou x=getcwd() renvoie dans x le répertoire courant de Scilab.

EXEMPLES

    pwd
x=pwd()

VOIR AUSSI

    chdir unix
