Scilab Function

NOM

    lstcat - concaténation de listes

SEQUENCE D'APPEL

lc=lstcat(l1,..ln)

PARAMETRES

    • li : une liste ou une variable de type indifférent
    • lc : une liste

DESCRIPTION

    lc=lstcat(l1,..ln) concatène les composantes des listes li sous forme d'une liste unique. Si les li sont des variables d'un autre type elles sont tout simplement ajoutées comme termes supplémentaires dans la liste.

EXEMPLES

    lstcat(list(1,2,3),33,list('foo',%s))
lstcat(1,2,3)

VOIR AUSSI

    list
