Scilab Function

NOM

    varn - indéterminée d'un polynôme

SEQUENCE D'APPEL

[symb]=varn(p)
[pm]=varn(x,var)

PARAMETRES

    • p : polynôme (ou matrice de polynômes)
    • symb : chaîne de caractères
    • x : polynôme (ou matrice de polynômes)
    • var : indéterminée du polynôme
    • pm : polynôme (ou matrice de polynômes)

DESCRIPTION

    symb=varn(p) renvoie dans symb l'indéterminée du polynôme p (par exemple varn(poly(0,'x')) est égal à 'x' ).

    varn(x,'s') renvoie une matrice polynomiale de même taille que x mais avec 's' comme indéterminée.

EXEMPLES

    s=poly(0,'s');p=[s^2+1,s];
varn(p)
varn(p,'x')
