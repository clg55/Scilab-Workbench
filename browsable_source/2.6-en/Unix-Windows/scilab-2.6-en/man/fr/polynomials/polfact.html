Scilab Function

NOM

    polfact - facteurs minimaux

SEQUENCE D'APPEL

[f]=polfact(p)

PARAMETRES

    • p : polynôme
    • f : vecteur [f0 f1 ... fn] tel que p=prod(f)
      • f0 : constante
      • fi : polynôme

DESCRIPTION

    renvoie les facteurs minimaux de p i.e. f=[f0 f1 ... fn] tel que p=prod(f)

VOIR AUSSI

    lcm cmndred factors
