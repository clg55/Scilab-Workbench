Scilab Function

NOM

    simp_mode - mode de simplification des fractions rationnelles

SEQUENCE D'APPEL

mod=simp_mode()
simp_mode(mod)

PARAMETRES

    • mod : un booléen

DESCRIPTION

    Une simplification rationnelle est effectuée après chaque opération sur les fractions rationnelles. Il est possible de changer cela avec la fonction simp_mode .

    simp_mod(%t) passe en mode de simplification systématique.

    simp_mod(%f) empêche les simplifications.

    mod=simp_mod() renvoie dans mod le mode de simplification courant.

EXEMPLES

    s=poly(0,'s');
mod=simp_mode()
simp_mode(%f);hns=s/s
simp_mode(%t);hns=s/s
simp_mode(mod);
