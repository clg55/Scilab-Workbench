Scilab function

NOM

    save_graph - sauve un graphe

SEQUENCE D'APPEL

save_graph(g,chemin)

PARAMETRES

    • g : graphe (liste)
    • chemin : chaîne de caractères, nom de fichier

DESCRIPTION

    save_graph sauve le graphe g dans un fichier. chemin est le nom du fichier où le graphe est sauvé. chemin peut être le nom du fichier ou le nom d'un répertoire; si l'extension "graph" est omise dans name , elle est ajoutée. Si chemin est le nom d'un répertoire, le nom du graphe est utilisé comme nom de fichier.

EXEMPLES

    g=load_graph(SCI+'/demos/metanet/mesh100');
show_graph(g);
unix('rm mymesh100.graph')
save_graph(g,'mymesh100.graph');
g=load_graph('mymesh100');
show_graph(g,'new');
