Scilab function

NOM

    load_graph - charge un graphe

SEQUENCE D'APPEL

g = load_graph(name)

PARAMETRES

    • name : chaîne de caractères, le chemin du graphe à charger
    • g : graphe (liste)

DESCRIPTION

    name est le nom d'un fichier contenant la description ASCII d'un graphe. Un tel fichier doit avoir l'extension "graph" . name peut être le nom ou le chemin du fichier; si l'extension "graph" est omise dans name , elle est ajoutée. load_graph renvoie le graphe correspondant.

EXEMPLES

    g=load_graph(SCI+'/demos/metanet/mesh100.graph');
show_graph(g);
g=load_graph(SCI+'/demos/metanet/colored');
show_graph(g,'new');
