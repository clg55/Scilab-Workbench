Scilab Function

NOM

    derivative - approximation de dérivées

SEQUENCE D'APPEL

J=derivative(f,x0,h)
[J,J2]=derivative(f,x0,h)

PARAMETRES

    • f : fonction Scilab f: R^n --> R^p
    • x0 : vecteur colonne réel de taille n
    • h : scalaire positif (1.d-7 par défaut)
    • J : matrice réelle p x n (approximation de la Jacobienne)
    • J2 : matrice réelle p x (p*n)

DESCRIPTION

    Dérivées approchées de la fonction f: R^n --> R^p . 

    f(x) = f(x0) + D1f(x0) *dx + 1/2* D2f(x0) *(dx.*.dx) +...
J = D1f(x0) ; J2=D2f(x0)

    Les dérivées sont approchées par différences finies : f_i(x+h)-f_i(x))/h. Attention : h doit être choisi avec précaution ...

EXEMPLES

    deff('y=f(x)','y=[sin(x(1))*cos(x(2));x(1)^2*x(2)^2;1+x(1)*x(2)^2]')
x0=[1;2];eps=0.001;h=eps*[2;5];
[J,J2]=derivative(f,x0);
[f(x0+h)-f(x0)-J*h ,0.5*J2*(h.*.h)]
