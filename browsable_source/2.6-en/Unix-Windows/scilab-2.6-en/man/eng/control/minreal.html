Scilab Function

NAME

    minreal - minimal balanced realization

CALLING SEQUENCE

slb=minreal(sl [,tol])

PARAMETERS

    • sl,slb : syslin lists
    • tol : real (threshold)

DESCRIPTION

    [ae,be,ce]=minreal(a,b,c,domain [,tol]) returns the balanced realization of linear system sl ( syslin list).

    sl is assumed stable.

    tol threshold used in equil1 .

EXAMPLES

    A=[-eye(2,2),rand(2,2);zeros(2,2),-2*eye(2,2)];
B=[rand(2,2);zeros(2,2)];C=rand(2,4);
sl=syslin('c',A,B,C);
slb=minreal(sl);
ss2tf(sl)
ss2tf(slb)
ctr_gram(sl)
clean(ctr_gram(slb))
clean(obs_gram(slb))
