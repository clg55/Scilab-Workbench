Scilab Function

NAME

    histplot - plot a histogram

CALLING SEQUENCE

histplot(npoint,data,[style,strf,leg,rect,nax])

PARAMETERS

    • npoint : integer or a row vector of increasing values.
    • data : real vector.
    • style,strf,leg,rect,nax : see plot2d .

DESCRIPTION

  • - If is an integer, plots a histogram of the values stored in using equally spaced classes.
  • - If is a vector plots a histogram of the values stored in using the classes .

    • Enter the command histplot() to see a demo.
