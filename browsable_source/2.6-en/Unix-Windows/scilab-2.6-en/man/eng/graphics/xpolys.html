NAME

    xpolys - draw a set of polylines or polygons

CALLING SEQUENCE

xpolys(xpols,ypols,[draw])

PARAMETERS

    • xpols,ypols : matrices of the same size (p,n) (points of the polylines).
    • draw : vector of size n.

DESCRIPTION

    xpolys draws a set of polylinse using marks or dashed lines. The coordinates of each polyline are stored in a column of xpols and ypols .

    The style of polyline i is given by draw(i) :

  • - If is negative, the mark with id is used to draw polyline i (marks are drawn using the current pattern). Use to see the meaning of the ids.
  • - If is strictly positive, the line style (or color) with id is used to draw polyline i. Use to see the meaning of the ids.

EXAMPLES

    plot2d(0,0,-1,"012"," ",[0,0,1,1])
rand("uniform")
xset("color",3)
xpolys(rand(3,5),rand(3,5),[-1,-2,0,1,2])
xset("default")
