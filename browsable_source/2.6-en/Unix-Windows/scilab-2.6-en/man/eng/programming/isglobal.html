Scilab Function

NAME

    isglobal - check if a variable is global

CALLING SEQUENCE

t=isglobal(x)

PARAMETERS

    • x : any variable
    • t : a boolean

DESCRIPTION

    isglobal(x) returns true if x has been declared to be a global variable and false otherwise.

EXAMPLES

    isglobal(1)
global a
isglobal(a)
