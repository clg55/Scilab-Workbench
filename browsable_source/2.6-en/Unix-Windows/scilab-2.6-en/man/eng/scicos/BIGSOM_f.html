Scicos Block

NAME

    BIGSOM_f - Scicos addition block

DESCRIPTION

    This block realize weighted sum of the input vectors. The output is vector k th component is the sum of the k th components of each input ports weighted by sgn(k) .

DIALOGUE PARAMETERS

  • Input signs : a vector of weights (generaly +1 or -1). The number of input signs fix the number of input ports.
