Scilab Function

NAME

    cmndred - common denominator form

CALLING SEQUENCE

[n,d]=cmndred(num,den)

PARAMETERS

    • num, den : two polynomial matrices of same dimensions

DESCRIPTION

    computes a polynomial matrix n and a common denominator polynomial d such that:

    n/d=num./den

    The rational matrix defined by num./den is n/d
