Scilab Function

NAME

    floor - rounding down

CALLING SEQUENCE

[y]=floor(x)

PARAMETERS

    • x : a real matrix
    • y : integer matrix

DESCRIPTION

    floor(x) returns an integer matrix made of nearest rounded down integers.

EXAMPLES

    floor([1.9 -2.5])-[1,-3]
floor(-%inf)
x=rand()*10^20;floor(x)-x
