Scilab Function

NAME

    gammaln - The logarithm of gamma function.

CALLING SEQUENCE

y = gammaln(x)

PARAMETERS

    • x : real vector
    • y : real vector with same size.

DESCRIPTION

    gammaln(x) evaluates the logarithm of gamma function at all the elements of x , avoiding underflow and overflow. x must be real.

EXAMPLES

    gammaln(0.5)
