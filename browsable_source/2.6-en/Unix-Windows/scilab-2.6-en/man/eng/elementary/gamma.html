Scilab Function

NAME

    gamma - The gamma function.

CALLING SEQUENCE

y = gamma(x)

PARAMETERS

    • x : real vector
    • y : real vector with same size.

DESCRIPTION

    gamma(x) evaluates the gamma function at all the elements of x . x must be real.

    gamma(n+1) = n!

EXAMPLES

    gamma(0.5)
gamma(6)-prod(1:5)
