Scilab Function

NAME

    atanm - square matrix tangent inverse

CALLING SEQUENCE

[t]=atanm(x)

PARAMETERS

    • x : real or complex square matrix
    • t : real or complex square matrix

DESCRIPTION

    atanm(x) is the matrix arctangent of the matrix x. Result may be not reliable if x is not symmetric.

EXAMPLES

    tanm(atanm([1,2;3,4]))
