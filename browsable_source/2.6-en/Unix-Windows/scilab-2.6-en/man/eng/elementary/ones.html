Scilab Function

NAME

    ones - matrix made of ones

CALLING SEQUENCE

y=ones(m1,m2,...)
y=ones(x)
y=ones()

PARAMETERS

    • x,y : matrices
    • m1, m2,.. : integers

DESCRIPTION

    Returns a matrix made of ones.

  • ones(m1,m2) returns a matrix full of ones.
  • ones(m1,m2,..,mn) : creates a matrix full of ones.
  • ones(x) returns a matrix full of ones with the same size that .
  • ones(x) is also valid for a list.

    • Note that ones(3) is ones(a) with a=3 i.e it is NOT a 3x3 matrix!

    ones() is eqivalent to ones(1,1).

EXAMPLES

    ones(3)
ones(3,3)
ones(2,3,2)
