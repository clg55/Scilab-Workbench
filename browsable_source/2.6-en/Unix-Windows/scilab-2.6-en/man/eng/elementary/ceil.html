Scilab Function

NAME

    ceil - rounding up

CALLING SEQUENCE

[y]=ceil(x)

PARAMETERS

    • x : a real matrix
    • y : integer matrix

DESCRIPTION

    returns an integer matrix made of rounded up elements

EXAMPLES

    ceil([1.9 -2.5])-[2,-2]
ceil(-%inf)
x=rand()*10^20;ceil(x)-x
