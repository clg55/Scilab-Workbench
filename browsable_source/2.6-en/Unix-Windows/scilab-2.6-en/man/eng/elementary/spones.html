Scilab Function

NAME

    spones - sparse matrix

CALLING SEQUENCE

sp=spones(A)

PARAMETERS

    • A : sparse matrix
    • sp : sparse matrix

DESCRIPTION

    sp=spones(A) generates a matrix with the same sparsity structure as A , but with ones in the nonzero positions;

EXAMPLES

    A=sprand(10,12,0.1);
sp=spones(A)
B = A~=0
bool2s(B)
