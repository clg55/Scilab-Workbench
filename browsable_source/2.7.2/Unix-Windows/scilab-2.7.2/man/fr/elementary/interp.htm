Scilab Function

interp - interpolation

Sequence d'appel

[f0 [,f1 [,f2 [,f3]]]]=interp(xd,x,f,d)

Parametres

    • xd : vecteur réel
    • x,f,d : vecteurs réels renvoyés par la fonction splin
    • fi : vecteur réel (dérivées)

Description

Étant donnés les trois vecteurs (x,f,d) définissant une spline (voir splin) avec fi=S(xi), di = S'(xi) cette fonction évalue S (resp. S', S'', S''') aux points xd(i).

  • x : vecteur des xi (x(1) < x(2) < ...)
  • f : vecteur des S(xi)
  • d : vecteur des S'(xi)
  • f0 : vecteur contenant [S(xd(1),S(xd(2)),S(xd(3)),...]
  • f(1 2 3) : vecteur des dérivées premières, secondes et troisièmes de S aux points xd=[xd(1),xd(2),...], c'est à dire

    • f1 = [S'(xd(1)),S'(xd(2)),...]

    f2 = [S''(xd(1)),S''(xd(2)),...]
