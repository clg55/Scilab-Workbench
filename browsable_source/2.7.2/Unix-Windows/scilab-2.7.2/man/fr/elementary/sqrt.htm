Scilab Function

sqrt - racine carrée

Sequence d'appel

y=sqrt(x)

Parametres

    • x : vecteur ou matrice (réelle ou complexe)

Description

sqrt(x) contient les racines carrées des éléments de x. Le résultat est complexe si des composantes de x sont négatives.

Exemples

    sqrt([2,4])
sqrt(-1)
