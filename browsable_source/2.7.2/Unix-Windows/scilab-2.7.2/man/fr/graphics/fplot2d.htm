Scilab Function

fplot2d - dessin 2D d'une courbe définie par une fonction

Sequence d'appel

fplot2d(xr,f,[style,strf,leg,rect,nax])
fplot2d(xr,f,<opt_args>)

Parametres

    • xr : vecteur.
    • f : fonction Scilab du type y=f(x) ou routine liée dynamiquement donnée sous forme d'une chaîne de caractères.
    • style,strf,leg,rect,nax : voir plot2d
    • <opt_args> : voir plot2d

Description

fplot2d dessine une courbe définie par la fonction f. La courbe est approchée par interpolation linéaire entre les points (xr(i),f(xr(i)). Les valeurs de f(x) sont obtenues par feval(xr,f).

Taper fplot2d() pour voir une démonstration de la fonction.

Exemples

    deff("[y]=f(x)","y=sin(x)+cos(x)")
x=[0:0.1:10]*%pi/10;
fplot2d(x,f)
xbasc();
fplot2d(1:10,'parab')
