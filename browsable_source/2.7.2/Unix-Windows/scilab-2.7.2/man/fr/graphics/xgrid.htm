xgrid - ajoute une grille sur un dessin 2D

Sequence d'appel

xgrid([style])

Parametres

    • style : entier

Description

xgrid ajoute une grille sur un dessin 2D. style est le numéro de couleur de la grille. Utiliser xset() pour voir les numéros de couleurs.

Exemples

    x=[0:0.1:2*%pi]';
plot2d(sin(x))
xgrid(2)
