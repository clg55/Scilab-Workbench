Scilab Function

printing - impression des graphiques Scilab

Sequence d'appel

Blatexpr xscale yscale filename.ps
BEpsf filename.ps
Blpr "Title" filename1.ps filename2.ps ... filenamen.ps | lpr

Description

Les graphiques peuvent être sauvegardés avec la commande xbasimp. La commande : 

 xbasimp(xx,'des.ps',0)

sauve les graphiques enregistrés dans la fenêtre xx dans le fichier des.ps.xx. Ce fichier ne peut pas être envoyé directement vers une imprimante Postscript mais un ensemble de programmes Scilab ( dans le répertoire bin de Scilab ) sont à utiliser pour cela :

  • BEpsf : La commande BEpsf crée un fichier epsf (PostScript encapsulé) à partir du fichier des.ps.xx sous le nom des.epsf, ce fichier epsf peut être imprimé directement ou inclus dans un fichier PostScript.
  • Blatexpr : La commande Blatexpr crée un fichier epsf à partir du fichier des.ps.xx des.ps.xx 
    •   mv des.ps.xx des.ps
  Blatexpr 1.0 1.0 des.ps

    sous le nom des.epsf et un fichier LaTeX des.tex. Le fichier des.tex peut être inséré dans un document LaTeX pour inclure le fichier epsf ( le fichier postscript est inclus avec la commande special de TeX ) 

     \input des.tex 
 \dessin{caption}{label}
  • Blpr : La commande Blpr est utilisée pour imprimer un ensemble de graphiques sur une même feuille de papier. Par exemple pour imprimer deux graphiques sur une seule page : 
    • Blpr "Deux graphiques" file1.ps.0 file2.ps.1 | lpr
  • Blatexprs : La commande Blatexprs utilisée pour insérer dans une figure LaTeX un ensemble de graphiques Scilab 
    • Blatexprs res file1.ps.0 file2.ps.1

    crée deux fichiers res.ps et res.tex. Le fichier res.tex est utilisé comme pour Blatexpr pour lire la figure.
