Scilab Function

edit_curv - éditeur graphique interactif

Sequence d'appel

[x,y,ok,gc] = edit_curv(y)
[x,y,ok,gc] = edit_curv(x,y)
[x,y,ok,gc] = edit_curv(x,y,job)
[x,y,ok,gc] = edit_curv(x,y,job,tit)
[x,y,ok,gc] = edit_curv(x,y,job,tit,gc)

Parametres

    • x : vecteur de coordonnées x
    • y : vecteur de coordonnées y
    • job : chaîne de caractères composée d'un à trois des caractères 'a','x','y'
      • 'a' : pour ajouter des points à la courbe éditée
      • 'x' : pour modifier les coordonnées x des points de la courbe éditée
      • 'y' : pour modifier les coordonnées y des points de la courbe éditée
    • tit : un vecteur de 3 chaînes de caractères donnant les légendes de la courbe
    • gc : une liste de paramètres pour la fenêtre graphique : gc=list(rect,nax)
      • rect : bornes du graphique (voir plot2d)
      • nax : paramètres de graduation (voir plot2d)
    • ok : si ok==%t l'utilisateur a quitté avec le menu 'ok' sinon l'utilisateur a quitté avec le menu 'abort'. : list (objets graphiques créés avec edit_curv

Description

edit_curv est un éditeur de courbes interactif. Pour ajouter un nouveau point cliquer à la position désirée, le point ajouté sera connecté au point final le plus proche. Pour bouger un point cliquer dessus, bouger la souris et cliquer pour définir la nouvelle position.
