Scilab Function

plot2d4 - représentation d'une courbe sous forme de flèches

Sequence d'appel

plot2d4([logflag,] x,y,[style,strf,leg,rect,nax])
plot2d4(y)
plot2d4(x,y <,opt_args>)

Parametres

    [ ] : voir plot2d pour une description des paramètres.

Description

plot2d4 est identique à plot2d mais les courbes sont dessinées en utilisant des flèches. Ceci est utilisé par exemple pour représenter la solution d'une équation différentielle dans l'espace des phases.

Par défaut, les dessins successifs sont superposés. Pour effacer le dessin précédent, utiliser xbasc().

Taper plot2d4() pour voir une démonstration de la fonction.

Exemples

    // comparer avec plot2d
x=[0:0.1:2*%pi]';
plot2d4(x,[sin(x) sin(2*x) sin(3*x)])
