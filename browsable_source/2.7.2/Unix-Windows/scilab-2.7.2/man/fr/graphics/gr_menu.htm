gr_menu - petit éditeur graphique interactif

Sequence d'appel

[sd1]=gr_menu([sd,flag,no_frame])

Parametres

    • sd : liste (sortie de gr_menu), ou vecteur de longueur 4 [xmin,ymin,xmax,ymax] donnant les bornes du dessin).
    • sd1 : liste (objets graphiques créés avec gr_menu
    • flag, noframe : entiers valant 0 ou 1. Utiliser flag=1 pour le mode non-interactif (i.e. pour redessiner des graphiques déjà créés avec gr_menu) et no_frame=1 pour supprimer le cadre autour des graphiques faits avec gr_menu.

Description

gr_menu est petit éditeur graphique interactif. Quand on exécute gr_menu(), 3 nouveaux menus, Objects, Settings et Edit sont ajoutés à la fenêtre graphique courante. Utiliser l'article Exit du menu Edit pour sortir de gr_menu.

Les graphiques sont sauvés sous forme d'une liste pouvant être donnée en entrée de gr_menu.

  • [sd]=gr_menu([xmin,ymin,xmax,ymax]) : lance gr_menu avec un cadre donné
  • [sd]=gr_menu(); : lance gr_menu avec le cadre par défaut [0 0 100 100].
  • [sd]=gr_menu(sd) : redessine les graphiques stockés dans sd et entre en mode interactif.
  • [sd]=gr_menu(sd,1) : redessine uniquement les graphiques stockés dans sd.
  • [sd]=gr_menu(sd,1,1) : redessine uniquement les graphiques stockés dans sd et aucun cadre n'est dessiné.
