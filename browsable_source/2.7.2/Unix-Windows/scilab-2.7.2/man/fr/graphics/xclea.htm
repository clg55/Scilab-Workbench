Scilab Function

xclea - efface un rectangle

Sequence d'appel

xclea(x,y,w,h)

Parametres

    • x,y,w,h : valeurs réelles définissant le rectangle.

Description

xclea efface le rectangle [x,y,w,h] (point en haut à gauche, largeur, hauteur) dans l'échelle graphique courante.

Exemples

    x=[0:0.1:2*%pi]';
plot2d(x,sin(x))
xclea(1,1,1,1)
