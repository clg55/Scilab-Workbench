param3d1 - représentation de courbes paramétriques dans l'espace

Sequence d'appel

param3d1(x,y,z,[theta,alpha,leg,flag,ebox])
param3d1(x,y,list(z,colors),[theta,alpha,leg,flag,ebox])

Parametres

    • x,y,z : matrices de même taille (nl,nc). Chaque colonne i des matrices correspond aux coordonnées des points de la courbe numéro i. On peut donner une couleur à chaque courbe en utilisant list(z,colors) au lieu de l'argument z, où colors est un vecteur de taille nc. Si color(i) est négatif la courbe est dessinée en utilisant le marqueur de numéro abs(style(i))+1; si style(i) est strictement positif, une ligne de couleur style(i) ou de code de pointillé style(i) est utilisée. Utiliser xset() pour voir les codes de couleurs et de marqueurs.
    • Pour les autres paramètres : voir

Description

param3d1 est utilisée pour dessiner dans l'espace des courbes paramétriques données par les points de coordonnées contenues dans les vecteurs x, y et z.

Taper param3d1() pour voir une démonstration de la fonction.

Exemples

    t=[0:0.1:5*%pi]';
param3d1([sin(t),sin(2*t)],[cos(t),cos(2*t)],..
  list([t/10,sin(t)],[3,2]),35,45,"X@Y@Z",[2,3])
