Scilab Function

champ - représentation d'un champ de vecteurs dans le plan

Sequence d'appel

champ(x,y,fx,fy,[arfact,rect,strf])
champ(x,y,fx,fy,<opt_args>)

Parametres

    • x,y : deux vecteurs définissant la grille
    • fx : une matrice décrivant la composante x du champ de vecteurs. fx(i,j) est la composante x du champ de vecteurs au point (x(i),y(j)).
    • fy : une matrice décrivant la composante y du champ de vecteurs. fy(i,j) est la composante y du champ de vecteurs au point (x(i),y(j)).
    • <opt_args> : une suite d'affectations key1=value1, key2=value2, ... où key1,key2, ... peuvent prendre les valeurs : afact, rect, strf (voir ci-dessous).
    • arfact : un argument optionnel de type réel donnant le facteur d'échelle pour l'affichage des flèches (1.0 par défaut).
    • rect : un vecteur rect=[xmin,ymin,xmax,ymax], les limites du dessin.
    • strf : une chaîne de caractères de longueur 3 "xyz" ayant la même signification que le paramètre strf de plot2d. Le premier caractère x n'a pas d'effet avec champ.

Description

champ dessine un champ de vecteurs dans le plan avec des flèches colorées. La longueur des flèches dépend de l'intensité du champ.

Si la couleur des flèches doit être proportionnelle à l'intensité du champ, utiliser champ1.

Taper champ() pour voir une démonstration de la fonction.

Exemples

    // utilisation de rect pour les limites du dessin
champ(-5:5,-5:5,rand(11,11),rand(11,11),1,[-10,-10,10,10],"011")
// utilisation de  (x,y) pour récuperer les limites 
xbasc()
champ(-5:5,-5:5,rand(11,11),rand(11,11),2,[-10,-10,10,10],"021")
