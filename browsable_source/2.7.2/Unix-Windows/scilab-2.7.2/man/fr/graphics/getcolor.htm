Scilab Function

getcolor - sélection de couleurs dans la table de couleurs courante

Sequence d'appel

c=getcolor(title,[cini])
c=getcolor()

Parametres

    • title : chaîne de caractères, titre de la boîte
    • cini : vecteur des couleurs initialement choisies (xget("pattern") par défaut)
    • c : vecteur des codes des couleurs sélectionnées, ou [] si le bouton "Cancel" a été actionné

Description

getcolor ouvre une boîte de dialogue avec autant de "palettes" que de couleurs dans cini.
