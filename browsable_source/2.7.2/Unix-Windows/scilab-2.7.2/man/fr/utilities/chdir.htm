Scilab Function

chdir - change le répertoire courant de Scilab

Sequence d'appel

ierr=chdir('chemin')

Parametres

    • ierr : un entier, 1 si chdir a échoué, 0 sinon.

Description

Choisit 'chemin' comme répertoire courant de Scilab.

Exemples

    chdir(TMPDIR);
if MSDOS then
  unix_w("dir");
else
  unix_w("ls");
end
