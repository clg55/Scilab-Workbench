Link

To connect an output port to an input port, select first the Link menu item, then click on the output port, drag, click left on each intermediate points and finally click left on the input port.

To split a link, select first the Link menu item, then click left on the link where the split should be placed, drag, click left on each intermediate points and finally click left on the input port.

Right click cancel the link action

Only one link can go from and to a port. Link color can be changed directly by clicking on the link.

This menu remains active until user choose another one.
