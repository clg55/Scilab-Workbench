Event variable delay

Library

Events

Description

Similar to an event delay but the delay time is read on the regular input port.

Dialog Box

Set parameter of variable event delay
Initial event firing time (0 if absent) -1
  • Initial event firing time : time an intial activation is programmed on the output event port. If set to a value less than zero, no event is programmed.

Default properties

  • always active: no
  • direct-feedthrough: no
  • zero-crossing: no
  • mode: no
  • number/sizes of inputs: 1 / 1
  • number/sizes of outputs: 0 /
  • number/sizes of activation inputs: 1 / 1
  • number/sizes of activation outputs: 1 / 1
  • continuous-time state: no
  • discrete-time state: no
  • name of computational function: evtvardly
