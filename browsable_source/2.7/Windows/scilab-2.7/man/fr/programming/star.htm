Scilab Function

star (*) - opérateur de multiplication

Description

Multiplication. Signification habituelle. Valide pour les matrices réelles, complexes, polynomiales, rationnelles et pour les listes de type syslin (dans ce dernier cas "*" désigne une connection en série).

La multiplication élément par élément est notée x.*y. Si x ou y est un scalaire (matrice 1x1) .* est équivalent à *.

Le produit de Kronecker se note x.*.y
