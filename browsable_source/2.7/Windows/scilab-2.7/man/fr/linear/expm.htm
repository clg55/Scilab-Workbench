expm - exponentielle de matrice

Sequence d'appel

expm(X)

Parametres

    • X : matrice carrée réelle ou complexe

Description

Pour X une matrice carrée expm(X) est la matrice

expm(X) = I + X + X^2 /2 + ...

Le calcul fait appel à une bloc-diagonalisation préliminaire de X suivie d'une approximation de Padé sur chaque bloc.

Exemples

    X=[1 2;3 4]
expm(X)
logm(expm(X))
