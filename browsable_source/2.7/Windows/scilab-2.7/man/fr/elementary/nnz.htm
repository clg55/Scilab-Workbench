Scilab Function

nnz - nombre de termes non nuls dans une matrice

Sequence d'appel

n=nnz(X)

Parametres

    • X : matrice réelle ou complexe (pleine ou creuse)
    • n : entier, le nombre de termes non nuls

Description

nnz compte le nombre de termes non nuls dans une matrice pleine ou creuse.

Exemples

    sp=sparse([1,2;4,5;3,10],[1,2,3]);
nnz(sp)
a=[1 0 0 0 2];
nnz(a)
