Scilab Function

subf - soustraction symbolique

Sequence d'appel

["c"]=subf("a","b")

Parametres

    • "a","b","c" : strings

Description

Renvoie la chaîne de caractères c="a-b". Des simplifications triviales du type subf("0","a") et subf("1","2") sont effectuées.

Exemples

    subf('0','a')
subf('2','1')
subf('a','0')
