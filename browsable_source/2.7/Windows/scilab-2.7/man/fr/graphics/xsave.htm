Scilab Function

xsave - sauve le contenu d'une fenêtre graphique dans un fichier

Sequence d'appel

xsave(file-name,[win-num])

Parametres

    • file-name : chaîne de caractères, nom du fichier
    • win-num : entier, numéro de fenêtre. Sinon, la fenêtre courante est utilisée.

Description

xsave sauve les objets graphiques contenus dans la fenêtre graphique win-num dans le fichier binaire file-name. Les graphiques sont stockés dans un fichier indépendant de la plate-forme (en utilisant la librairie xdr) et peuvent être rechargés avec xload.
