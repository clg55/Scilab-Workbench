Scilab Function

hotcolormap - table de couleurs passant du rouge au jaune

Sequence d'appel

cmap=hotcolormap(n)

Parametres

    • n : entier >= 3, taille de la table
    • cmap : matrice à 3 colonnes [R,V,B].

Description

hotcolormap calcule une table de couleurs avec n couleurs "chaudes" allant du rouge au jaune.

Exemples

    xset("colormap",hotcolormap(32))
plot3d1()
