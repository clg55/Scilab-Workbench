champ1 - champ de vecteurs dans le plan avec flèches colorées

Sequence d'appel

champ1(x,y,fx,fy,[arfact,rect,strf])

Parametres

    • x,y : deux vecteurs définissant la grille
    • fx : une matrice décrivant la composante x du champ de vecteurs. fx(i,j) est la composante x du champ de vecteurs au point (x(i),y(j)).
    • fy : une matrice décrivant la composante y du champ de vecteurs. fy(i,j) est la composante y du champ de vecteurs au point (x(i),y(j)).
    • arfact : un argument optionnel de type réel donnant le facteur d'échelle pour l'affichage des flèches (1.0 par défaut).
    • rect : un vecteur rect=[xmin,ymin,xmax,ymax] les limites du dessin
    • strf : une chaîne de caractères de longueur 3 "xyz" ayant la même signification que le paramètre strf de plot2d. Le premier caractère x n'a pas d'effet avec champ1.

Description

champ1 dessine un champ de vecteurs dans le plan avec des flèches colorées. La couleur des flèches dépend de l'intensité du champ.

Si la longueur des flèches doit être proportionnelle à l'intensité du champ, utiliser champ.

Taper champ1() pour voir une démonstration de la fonction.

Exemples

    xset("use color",1)
champ1(-5:5,-5:5,rand(11,11),rand(11,11),2,[-10,-10,10,10],"021")
