xload - charge des objets graphiques enregistrés dans un fichier

Sequence d'appel

xload(file-name,[win-num])

Parametres

    • file-name : chaîne de caractères, nom du fichier
    • win-num : entier, numéro de fenêtre. Sinon, la fenêtre courante est utilisée.

Description

xload redessine les objets graphiques contenus dans le fichier file-name dans la fenêtre graphique win-num.
