GENSIN_f - Scicos sinusoid generator

Description

This block is a sine wave generator: M*sin(F*t+P)

DIALOGUE PARAMETERS

  • Magnitude : a scalar. The magnitude M.
  • Frequency : a scalar. The frequency F.
  • Phase : a scalar. The phase P.
