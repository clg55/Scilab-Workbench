Scilab Function

cmndred - dénominateur commun

Sequence d'appel

[n,d]=cmndred(num,den)

Parametres

    • num, den : 2 matrices de polynômes de mêmes dimensions

Description

calcule une matrice de polynômes n et un dénominateur commun d tels que :

n/d=num./den

La matrice de polynômes définie par num./den est n/d.
