Scilab Function

polfact - facteurs minimaux

Sequence d'appel

[f]=polfact(p)

Parametres

    • p : polynôme
    • f : vecteur [f0 f1 ... fn] tel que p=prod(f)
      • f0 : constante
      • fi : polynôme

Description

renvoie les facteurs minimaux de p i.e. f=[f0 f1 ... fn] tel que p=prod(f)
