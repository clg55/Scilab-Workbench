Scilab Function

derivat - dérivée d'une matrice de polynômes

Sequence d'appel

pd=derivat(p)

Parametres

    • p : polynôme, matrice de polynômes ou de fractions rationnelles

Description

Calcule la dérivée des polynômes ou fractions rationnelles contenus dans les termes de la matrice p.

Exemples

    s=poly(0,'s');
derivat(1/s)  // -1/s^2;
