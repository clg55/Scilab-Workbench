Scilab Function

semi (;) - instruction and row separator

Description

semicolons are used to separate rows in a matrix definition.

semicolons may also be used at the end of an instruction. In this case it means that the result(s) are not displayed. Conversely use comma (,) to get the display

Examples

    a=[1,2,3;4,5,6];
a=1;b=1,c=2
