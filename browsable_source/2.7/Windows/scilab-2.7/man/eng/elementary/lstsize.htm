Scilab Function

lstsize - list, tlist, mlist numbers of entries

Calling Sequence

n=lstsize(x)

Parameters

    • l : a list, tlist or mlist object
    • n : an integer, the number of entries

Description

lstsize(x) returns the number of entries for list, list, mlist objects. This function is more efficient than the size function and works similarily with all list types while size is overloaded for mlist objects.

Examples

    lstsize(list(1,'aqsdf'))
x=ssrand(3,2,4);
[ny,nu]=size(x)
lstsize(x)
