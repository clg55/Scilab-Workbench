edit_graph - graph and network graphical editor

Calling Sequence

num=edit_graph()
num=edit_graph(file_name [,zoom [,wsize]])
num=edit_graph(G [,zoom [,wsize]])

Parameters

    • file_name : character string. The path of a "graph" file
    • G : a graph-listdata structure
    • zoom : real positive scalar. The zoom factor, its default value is 1.
    • wsize : real positive vector [width height]. The initial window dimensions, its default value is [600,400].
    • num : integer scalar. The associated window number window dimension.

Description

This function starts a network graphical editor. Each time edit_graph is executed, a new editor window is created. The editor capabilities and menus are described in edit_graph_menus
