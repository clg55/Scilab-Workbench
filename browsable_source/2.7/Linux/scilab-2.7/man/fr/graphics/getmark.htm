Scilab Function

getmark - boîte de dialogue pour sélectionner un marqueur

Sequence d'appel

[mark,mrkSize]=getmark()

Parametres

    • mark : entier, le numéro du marqueur choisi
    • mrkSize : entier, la taille du marqueur choisi

Description

getmark ouvre une fenêtre graphique pour sélectionner un marqueur. Usage : [mark,mrkSize]=getmark(). xset("mark size",mrkSize). plot2d(x,y,style=mark).
