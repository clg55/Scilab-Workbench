Scilab Function

histplot - dessine un histogramme

Sequence d'appel

histplot(npoint,data,[style,strf,leg,rect,nax])
histplot(npoint,data,<opt_args>)

Parametres

    • npoint : entier ou vecteur ligne de valeurs croissantes
    • data : vecteur réel
    • style,strf,leg,rect,nax : voir plot2d.
    • <opt_args> : Représente une séquence d'instructions de la forme :key1=value1, key2=value2,... où key1, key2,... peuvent être :
      • normalization : Si sa valeur est %f les valeurs inscrites le long de l'axe des y sont les nombres d'elements Ni dans chaque intervalle, sinon les valeurs inscrites sont Ni/N/lbi où N est le nombre total d'éléments et lbi la longueur du lbième intervalle.
      • style : donne le style de chaque courbe (voir plot2d)
      • leg : donne la légende de chaque courbe. Si ce paramètre est spécifié et que strf n'est pas donné alors le caractère x de strf est supposé être 1 (voir plot2d)
      • rect : définit les bornes du dessin. Si ce paramètre est donné et que ni frameflag ni strf ne sont donnés alors le caractère y de strf est supposé être 7 (voir plot2d)
      • nax : définition des graduations. Si ce paramètre est donné et que ni axesflag ni strf ne sont donnés alors le caractère z de strf est supposé être 1 (voir plot2d)
      • logflag : définit le type d'échelle (linéaire ou logarithmique) des axes (voir plot2d)
      • frameflag : spécifie comment est calculé le cadre du dessin. Cette valeur est un entier entre 0 et 8. Il correspond au caractère y de strf (voir plot2d)
      • axesflag : spécifie le type d'axe dessiné autour du dessin. Cette valeur est un entier entre 0 et 5. Il correspond au caractère z de strf (voir plot2d)

Description

  • - Si npoint est un entier, histplot dessine un histograme des valeurs stockées dans data en utilisant npoint classes équiréparties.
  • - Si npoint est un vecteur histplot dessine un histograme des valeurs stockées dans data en utilisant les classes ]npoint(k),npoint(k+1)].

    • Exemples

      histplot()

d=rand(1,10000,'normal');
xbasc();histplot(20,d)

xbasc();histplot(20,d,normalization=%f)
xbasc();histplot(20,d,leg='rand(1,10000,''normal'')',style=5)
