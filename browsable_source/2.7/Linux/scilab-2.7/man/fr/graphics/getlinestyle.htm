Scilab Function

getlinestyle - boîte de dialogue pour sélectionner un style de ligne

Sequence d'appel

k=getlinestyle()

Parametres

    • k : entier, style de ligne sélectionné ou [] si le bouton "Cancel" a été actionné.

Description

getlinestyle ouvre une fenêtre graphique pour sélectionner un style de ligne. Exemple : k=getlinestyle() plot2d(1:10,10,style=k).
