coshm - cosinus hyperbolique matriciel

Sequence d'appel

t=coshm(x)

Parametres

    • x,t : matrice carrée réelle ou complexe

Description

coshm est le cosinus hyperbolique matriciel de la matrice x. t=(expm(x)+expm(-x))/2. Le résultat peut être imprécis pour les matrices non symétriques.

Exemples

    A=[1,2;2,4]
acoshm(coshm(A))
