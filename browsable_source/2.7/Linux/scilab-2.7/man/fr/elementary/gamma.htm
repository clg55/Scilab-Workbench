gamma - La fonction gamma.

Sequence d'appel

y = gamma(x)

Parametres

    • x : vecteur ou matrice de nombres réels.
    • y : vecteur ou matrice de nombre réels de même taille que x.

Description

gamma(x) évalue la fonction gamma pour les composantes de f x. x doit être à composantes réelles. La fonction gamma(x) est définie par:

Exemples

    // simple examples
gamma(0.5)
gamma(6)-prod(1:5)
// the graph of the Gamma function on [a,b]
a = -3; b = 5;
x = linspace(a,b,40000)';
y = gamma(x);
xbasc()
c=xget("color")
xset("color",2)
plot2d(x, y, style=0, axesflag=5, rect=[a, -10, b, 10])
xset("color",c)
fs=xget("font")
xset("font",4,2)
xtitle("The gamma function on ["+string(a)+","+string(b)+"]")
xset("font",fs(1),fs(2))
xselect()
