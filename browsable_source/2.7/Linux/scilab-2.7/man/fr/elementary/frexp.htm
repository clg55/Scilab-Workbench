Scilab Function

frexp - calcule la représentation en base 2 (mantisse et exposant) d'un nombre flottant.

Sequence d'appel

[f,e]=frexp(x)

Parametres

    • x : matrice ou vecteur réel
    • f : matrice ou vecteur réel, habituellement dans l'intervalle 0.5 <= abs(f) < 1.
    • e : matrice ou vecteur d'entiers satisfaisant l'équation: x = f.*2.^e

Description

Cette fonction correspond à la fonction ANSI C frexp(). Chaque 0 dans x produit f=0 et e=0.

Exemples

    [f,e]=frexp([1,%pi,-3,%eps])
