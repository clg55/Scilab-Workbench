Scilab Function

rdivf - division symbolique à droite

Sequence d'appel

["r"]=ldivf("d","c")

Parametres

    • "d","c","r" : strings

Description

Renvoie la chaîne de caractères "c/d" Des simplifications triviales du type "c/1" = "c" sont effectuées.

Exemples

    ldivf('c','d')
ldivf('1','2')
ldivf('a','0')
