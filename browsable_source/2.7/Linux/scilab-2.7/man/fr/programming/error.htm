Scilab Function

error - génération d'un message d'erreur

Sequence d'appel

error('string' [,n])
error(m)

Description

affiche la chaîne de caractères 'string' dans un message d'erreur et stoppe l'instruction courante.

Si n est donné, il précise le numéro de l'erreur. n doit être supérieur à 10000 (valeur par défaut).

error(m) affiche le message d'erreur numéro m.
