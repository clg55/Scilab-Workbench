Scilab Function

less (<) - comparaison, signe "inférieur à"

Description

Symbole de comparaison logique

  • <> signifie "différent de" (identique à ~=)
  • < signifie "inférieur à"
  • > signifie "supérieur à"
  • <= signifie "inférieur ou égal à"
  • >= signifie "supérieur ou égal à"
