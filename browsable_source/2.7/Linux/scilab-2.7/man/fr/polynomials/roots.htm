roots - racines d'un polynôme

Sequence d'appel

[x]=roots(p)

Parametres

    • p : polynôme à coefficients réels ou complexes

Description

x=roots(p) renvoie dans le vecteur complexe x les racines du polynôme p. Le degré de p doit être <=100.

Exemples

    p=poly([0,10,1+%i,1-%i],'x');
roots(p)
A=rand(3,3);roots(poly(A,'x'))    // comparaison via le polynôme caractéristique
spec(A)
