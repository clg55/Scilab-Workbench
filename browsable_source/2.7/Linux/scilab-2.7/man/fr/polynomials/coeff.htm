coeff - coefficients d'une matrice de polynômes

Sequence d'appel

[C]=coeff(Mp [,v])

Parametres

    • Mp : matrice de polynômes
    • v : vecteur (ligne ou colonne) d'entiers des degrés choisis
    • C : grosse matrice de coefficients

Description

C=coeff(Mp) renvoie dans une grosse matrice C les coefficients de la matrice de polynômes Mp. C est partitionnée comme
