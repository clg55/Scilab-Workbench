Scilab Function

getd - chargement de toutes les fonctions contenues dans un répertoire

Sequence d'appel

getd(path)

Parametres

    • path : chaîne de caractères, chemin du répertoire

Description

Charge tous les fichiers .sci (contenant des fonctions Scilab) présents dans le répertoire path.

Exemples

    getd('SCI/macros/auto')
