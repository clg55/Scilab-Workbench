Scilab Function

quote (') - opérateur de transposition, délimiteur de chaîne de caractères

Description

L'apostrophe ou "quote" (') est utilisé pour noter la transposition (conjuguée) des matrices.

Le symbole (.') doit être utilisé pour une transposition non conjuguée.

L'apostrophe (') ou le guillemet (") sont aussi utilisés pour délimiter les chaînes de caractères (une chaîne de caractères est définie entre deux apostrophes ou guillemets). Une apostrophe ou un guillemet doivent être doublés s'ils font partie de la chaîne de caractères elle même.

Exemples

    [1+%i, 2]'
[1+%i, 2].'
x='Ceci est une chaîne de caractères'
'Et Dieu dit : ''que la lumière soit'''
