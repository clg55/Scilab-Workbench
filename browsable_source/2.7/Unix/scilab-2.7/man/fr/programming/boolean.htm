Scilab Objects

boolean - Objets Scilab, variables booléennes et opérateurs & | ~

Description

Une variable booléenne est %T (pour "vrai"="true") ou %F (pour "faux"). Ces variables permettent de définir des matrices de booléens avec la syntaxe habituelle. Ces matrices booléennes peuvent être manipulées comme des matrices ordinaires pour les opérations d'extraction/insertion et de concaténation. Notez que les autres opérations usuelles (+, *, -, ^, etc.) ne sont pas définies pour les matrices booléennes. Les seuls opérateurs valides sont les suivants :

  • ~b : est la négation logique élément par élément de la matrice booléenne b.
  • b1&b2 : est la conjonction (et) logique élément par élément de b1 et b2 (matrices).
  • b1|b2 : est la disjonction (ou) logique élément par élément de b1 et b2 (matrices).
  • Les variables booléennes peuvent être utilisées comme indice pour les matrices et les vecteurs. Par exemple a([%T,%F,%T],:) renvoie la sous-matrice composée des lignes 1 et 3 de a. Les matrices booléennes peuvent être creuses.

    • Exemples

      [1,2]==[1,3]
[1,2]==1
a=1:5; a(a>2)
