Scilab Function

dft - discrete Fourier transform

Sequence d'appel

[xf]=dft(x,flag);

Parametres

    • x : input vector
    • flag : indicates dft (flag=-1) or idft (flag=1)
    • xf : output vector

Description

Function which computes dft of vector x.

Exemples

    n=8;omega = exp(-2*%pi*%i/n);
j=0:n-1;F=omega.^(j'*j);  //Fourier matrix
x=1:8;x=x(:);
F*x
fft(x,-1)
dft(x,-1)
inv(F)*x
fft(x,1)
dft(x,1)
