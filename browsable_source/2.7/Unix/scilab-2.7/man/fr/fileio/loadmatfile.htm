Scilab Function

loadmatfile - Charge un ficher .mat Matlab dans Scilab

Sequence d'appel

loadmatfile(filename)

Parametres

    • filename : Chaîne de caractères contenant le chemin du fichier

Description

Charge un ficher .mat Matlab dans Scilab. Les types de données Matlab sont convertis en leur équivalent Scilab.
