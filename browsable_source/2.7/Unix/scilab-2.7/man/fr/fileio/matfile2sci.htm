Scilab Function

matfile2sci - convertit un fichier Matlab .mat en un fichier binaire Scilab

Sequence d'appel

matfile2sci(mat_file_path,sci_file_path)

Parametres

    • mat_file_path : Chaîne de caractères contenant le chemin du fichier d'entrée
    • sci_file_path : Chaîne de caractères contenant le chemin du fichier résultant

Description

convertit un fichier Matlab .mat en un fichier binaire compatible avec la fonction . Les types de données Matlab sont convertis en leur équivalent Scilab.
