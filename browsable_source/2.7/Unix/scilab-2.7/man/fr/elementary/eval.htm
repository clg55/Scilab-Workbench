Scilab Function

eval - évaluation d'une matrice de chaînes de caractères

Sequence d'appel

[H]= eval(Z)

Description

renvoie le résultat de l'évaluation d'une matrice de chaînes de caractères Z.

Exemples

    a=1; b=2; Z=['a','sin(b)'] ; eval(Z)  // renvoie la matrice [1,0.909];
