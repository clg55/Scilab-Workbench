Scilab Function

floor - partie entière

Sequence d'appel

[y]=floor(x)

Parametres

    • x : matrice réelle
    • y : matrice entière

Description

Renvoie dans y pour chaque élément de x sa partie entière (l'entier directement inférieur).

Exemples

    floor([1.9 -2.5])-[1,-3]
floor(-%inf)
x=rand()*10^20;floor(x)-x
