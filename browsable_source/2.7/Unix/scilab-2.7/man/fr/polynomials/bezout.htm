Scilab Function

bezout - équation de Bezout pour les polynômes

Sequence d'appel

[pgcd,U]=bezout(p1,p2)

Parametres

    • p1, p2 : deux polynômes réels

Description

calcule le PGCD pgcd de p1 et p2 et de plus une matrice unimodulaire (2x2) U telle que :

[p1,p2]*U = [pgcd,0]

Le PPCM de p1 et p2 est donné par

p1*U(1,2) (ou -p2*U(2,2))

Exemples

    x=poly(0,'x');
p1=(x+1)*(x-3)^5;p2=(x-2)*(x-3)^3;
[pgcd,U]=bezout(p1,p2) 
det(U)
clean([p1,p2]*U)
ppcm=p1*U(1,2)
lcm([p1,p2])
