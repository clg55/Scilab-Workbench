plot2d2 - représentation dans le plan de fonctions constantes par morceaux

Sequence d'appel

plot2d2([x],y)
plot2d2([x],y,<opt_args>)
plot2d2([logflag],x,y,[style,strf,leg,rect,nax])

Parametres

    [ ] : voir plot2d pour une description des paramètres.

Description

plot2d2 est identique à plot2d mais les fonctions (x,y) sont supposées être constantes par morceaux.

Par défaut, les dessins successifs sont superposés. Pour effacer le dessin précédent, utiliser xbasc().

Taper plot2d2() pour voir une démonstration de la fonction.

Exemples

    // fonction valant i sur le segment [i,i+1]
// le dernier segment n'est pas dessiné
plot2d2([1:4],[1:4],1,"111","step function",[0,0,5,5])
// comparaison avec plot2d
x=[0:0.1:2*%pi]';
xbasc()
plot2d2(x,[sin(x) sin(2*x) sin(3*x)])
