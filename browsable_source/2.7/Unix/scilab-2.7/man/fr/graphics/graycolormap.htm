Scilab Function

graycolormap - table de couleurs allant du noir au blanc

Sequence d'appel

cmap=graycolormap(n)

Parametres

    • n : entier >= 1, taille de la table
    • cmap : matrice à 3 colonnes [R,V,B].

Description

graycolormap calcule une table de couleurs avec n nuances de gris variant linéairement du noir au blanc.

Exemples

    xset("colormap",graycolormap(32))
plot3d1()
