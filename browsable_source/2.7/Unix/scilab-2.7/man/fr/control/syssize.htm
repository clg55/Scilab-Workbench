Scilab Function

syssize - size of state-space system

Sequence d'appel

[r,nx]=syssize(Sl)

Parametres

    • Sl : linear system (syslin list) in state-space
    • r : 1 x 2 real vector
    • nx : integer

Description

returns in r the vector [number of outputs, number of inputs] of the linear system Sl. nx is the number of states of Sl.
