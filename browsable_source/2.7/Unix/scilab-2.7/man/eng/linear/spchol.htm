Scilab Function

spchol - sparse cholesky factorization

Calling Sequence

[R,P] = spchol(X)

Parameters

    • X : symmetric positive definite real sparse matrix
    • P : permutation matrix
    • R : cholesky factor

Description

[R,P] = spchol(X) produces a lower triangular matrix R such that P*R*R'*P' = X.
