Constant

Library

Sources

Description

This block is a constant value(s) generator.

Dialog Box

Set Contant Block
Constant 1
  • Constant: a real vector. The vector size gives the size of the output port. The value constants(i) is assigned to the ith component of the output.

Default properties

  • always active: no
  • direct-feedthrough: no
  • zero-crossing: no
  • mode: no
  • number/sizes of inputs: 0 /
  • number/sizes of outputs: 1 / 1
  • number/sizes of activation inputs: 0 /
  • number/sizes of activation outputs: 0 /
  • continuous-time state: no
  • discrete-time state: no
  • name of computational function: cstblk
