Eval

All dialogs user answers may be Scilab instructions they are evaluated immediately and stored as character strings. select this menu item to have them re-evaluated according to new values of underlying Scilab variables.

These underlying Scilab variables may be user global variables defined before Scicos was launch, They may also be defined in by the Scicos context (see Context menu item).
